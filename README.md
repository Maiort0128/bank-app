# Banking App

A simple console-based banking application written in Java. Supports account creation, deposits, withdrawals, transfers, and transaction history.

## Features
- Create new bank accounts (auto-generated account numbers)
- Deposit and withdraw money
- Transfer funds between accounts
- View transaction history per account
- Find accounts by account number or owner name
- Input validation (minimum deposit, empty name protection)

## How to Run

### Prerequisites
- Java 17 or higher installed

## What I Learned

- HashMap for O(1) account lookups
- Separation of concerns (Account/Bank/Main)
- Input validation and error handling
- Transaction history with ArrayList
